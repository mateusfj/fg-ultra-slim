# GF Ultra Slim - Landing Page

Landing page desenvolvida em **Next.js** para o produto GF Ultra Slim.

## Tecnologias utilizadas

- **Next.js 15** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS 4**
- **ESLint** (config Next.js)
- **PostCSS**
- **Radix UI** (componentes de interface acessíveis)
- **Embla Carousel** (carrosséis)

## Pré-requisitos

- **Node.js** 18+ instalado
- **npm** (ou outro gerenciador compatível, como pnpm/yarn)

## Como instalar as dependências

Dentro da pasta do projeto:

```bash
npm install
```

## Como rodar o projeto em desenvolvimento

```bash
npm run dev
```

O projeto ficará disponível (por padrão) em:

```bash
http://localhost:3000
```

## Como gerar build de produção

```bash
npm run build
```

E para rodar o build de produção localmente:

```bash
npm start
```
